# Milestone v5.0 Requirements Archive: Electromagnetism & Circuits

**Shipped:** 2026-06-11
**Milestone:** v5.0

## Completed Requirements

- [x] **EM-01**: User can add, select, and drag positive ($+q$) and negative ($-q$) point charges on the canvas. (Outcome: Validated)
- [x] **EM-02**: Sandbox displays vector arrows on a grid representing electric field intensity and direction at discrete points. (Outcome: Validated)
- [x] **EM-03**: Sandbox displays field lines originating from positive charges and terminating on negative charges or extending to infinity. (Outcome: Validated)
- [x] **EM-04**: Sandbox renders equipotential curves as smooth isolines representing constant electric potential. (Outcome: Validated)
- [x] **EM-05**: Particle gun component launches charged test particles into uniform magnetic field regions. (Outcome: Validated)
- [x] **EM-06**: Test particles trace circular, helical, or cycloidal paths depending on velocity vector and field parameters. (Outcome: Validated)
- [x] **EM-07**: Solver engine computes dynamic loop equation values (currents, voltages) for resistor ($R$), capacitor ($C$), and inductor ($L$) networks. (Outcome: Validated)
- [x] **EM-08**: Solver supports interactive switch component clicks to open/close loops. (Outcome: Validated)
- [x] **EM-09**: GraphModule renders transient charging curves, AC phase shifts, and resonant waveforms from virtual voltage/current probes. (Outcome: Validated)
