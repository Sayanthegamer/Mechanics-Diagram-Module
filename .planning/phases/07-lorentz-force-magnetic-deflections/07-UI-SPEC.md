## Visuals

| Property | Value |
|----------|-------|
| Primary Focal Point | Dual canvas viewport showing point charges, the interactive draggable particle gun turret, launched particles tracing Lorentz force trajectories, and the background uniform magnetic field. |
| Icon-Only Action Accessibility | Explicit `aria-label` attribute on all interactive icon buttons (e.g., visual mode toggles, fire button) paired with hover/focus tooltips displaying the action name. |
| Visual Hierarchy | High-contrast glowing point charges ($+q$ in red, $-q$ in blue) and the purple-accented particle gun launcher turret rendered on top of the dark canvas background. Launched particles trace glowing yellow/green solid trails. The uniform B-field background symbols ($\times$ and $\bullet$) or lines are drawn with low contrast (`rgba`) in the background layer to avoid visual clutter. |

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0c0c0e | App background, main canvas backdrop |
| Secondary (30%) | #16161a | Sidebar backdrop, control card backgrounds |
| Accent (10%) | #6366f1 | Active selection outline, button highlights |
| Destructive | #ef4444 | Reset buttons, error state text |

Accent reserved for:
- Highlighted/selected charge or gun turret boundary
- Hover and focus states of primary UI action controls
- Selected parameter value highlights in the telemetry list

Special Palette Mapping:
- Particle Gun Turret: `#a855f7` (Purple accent)
- Particle Trajectory Trail: `#eab308` (Glowing yellow solid trail)
- Magnetic Field Symbols/Lines: `rgba(255, 255, 255, 0.15)` (Semi-transparent white in dark mode)
- Equipotential Contours: `rgba(20, 184, 166, 0.4)` (Teal/green semi-transparent isolines)
