# Plan 2: Implement Lorentz Numerical Integration, Particle Trails, B-field rendering, Collision Absorption, and Telemetry

## <objective>
Finish Phase 7 by implementing the physics engine for moving test particles:
1. Implement the RK4 integration loop to step particle coordinates under combined static $E$-fields and uniform $B$-fields.
2. Render B-field symbols ($\times$ inside circles for into page, $\bullet$ inside circles for out of page) or actual vector grid lines underneath charges.
3. Draw solid trajectory trails for active test particles (yellow/green solid trails).
4. Implement contact absorption/annihilation when a test particle hits a static charge.
5. Wire real-time telemetry (speed, coords, kinetic energy) to the `GraphModule`.
</objective>
