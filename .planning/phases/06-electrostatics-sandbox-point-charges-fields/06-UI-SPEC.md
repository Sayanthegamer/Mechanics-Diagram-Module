## Visuals

| Property | Value |
|----------|-------|
| Primary Focal Point | Dual canvas viewport showing point charges, electric field lines, and equipotential contours |
| Icon-Only Action Accessibility | Explicit `aria-label` attribute on all interactive icon buttons (e.g., delete buttons, control toggles) paired with hover/focus tooltips displaying the action name |
| Visual Hierarchy | High-contrast, glowing point charges ($+q$ in red, $-q$ in blue) rendered on top of the dark canvas background draw immediate attention, while active selections are highlighted with a glowing dashed accent-color circle. Control sidebars are visually nested using a muted background hierarchy. |

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0c0c0e | App background, main canvas backdrop |
| Secondary (30%) | #16161a | Sidebar backdrop, control card backgrounds |
| Accent (10%) | #6366f1 | Active selection outline, button highlights |
| Destructive | #ef4444 | Charge deletion buttons, error state text |

Accent reserved for:
- Highlighted/selected charge boundary (glowing dashed circle on canvas)
- Hover and focus states of primary UI action controls
- Selected parameter value highlights in the telemetry list

Special Palette Mapping:
- Positive Charges: `#ef4444` (Red color indicating positive point charge $+q$)
- Negative Charges: `#3b82f6` (Blue color indicating negative point charge $-q$)
- Equipotential Contours: `rgba(20, 184, 166, 0.4)` (Teal/green semi-transparent isolines)
