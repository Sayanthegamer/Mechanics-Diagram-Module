## Visuals

| Property | Value |
|----------|-------|
| Primary Focal Point | The dynamic interactive schematic grid canvas displaying components, voltage gradients, and current particles. |
| Icon-Only Action Accessibility | Explicit `aria-label` attribute on all interactive icon buttons (such as canvas control toolbars or switch toggle button nodes) paired with hover/focus tooltips displaying the action name. |
| Visual Hierarchy | The schematic breadboard canvas acts as the main canvas workspace. Wires, nodes, and breadboard rails are high contrast, colored by electrical potential (positive, neutral, negative). Active components or wires selected for parameter adjustments show a glowing indigo border or dash ring. |

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0c0c0e | Background, main surfaces (Dark Theme) |
| Secondary (30%) | rgba(22, 22, 26, 0.75) | Cards, sidebars, header backdrop |
| Accent (10%) | #6366f1 | Highlight outlines, cursor pointer drag anchors |
| Destructive | #ef4444 | Destructive actions, reset states |
| Positive Potential | #ef4444 | Wires/nodes at V > 0V (bright red) |
| Neutral Potential | #505050 | Wires/nodes at V = 0V (green/gray) |
| Negative Potential | #3b82f6 | Wires/nodes at V < 0V (bright blue) |
| Current Particle | #fbbf24 | Moving flow dot markers (Golden amber) |

Accent reserved for: Active component canvas selection glow, focus state borders, range slider thumb hover rings.
