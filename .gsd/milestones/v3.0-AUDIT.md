## Concerns
- **Floating Point Stability**: Real-time integration (RK4/Verlet) is stable for long runs but extremely high gravity parameters or small orbits could theoretically still hit integration limits. Trail capping limits (400-500 coordinates) keep memory leaks at zero.
- **Canvas Resize**: High-DPI screen resizing has been thoroughly checked. Canvases scale correctly without losing tracking state.

## Recommendations
1. **Verlet Solver for Carnot Piston (Thermodynamics)**: Apply the Verlet numerical integration lessons from Two-Body gravity directly to the Piston engine physics to guarantee long-term thermodynamic stability.
2. **Continue Capped Array Paradigm**: Ensure the 200-point history capping checked during Phase 4 is preserved in the next milestone when plotting entropy and distribution histograms.
